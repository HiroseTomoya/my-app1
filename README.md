<多機能メモツール>


概要
　本アプリは、日常のタスク管理やメモ管理を効率化することを目的として開発した、多機能型のデスクトップアプリケーションです。 Python（Tkinter）を用いて、複数の機能を一つのアプリに統合しました。

主な機能

　⏲ タイマー機能 📝 ToDoリスト 📄 メモ（フォルダ分け対応） 📅 カレンダー（日付ごとのメモ管理） 🔒 セキュリティ強化メモ（パスワード保護）

工夫した点

　① 並行処理の実装

　　タイマー機能を使用しながら他の機能を同時に操作できるように、threadingを用いた並行処理を実装しました。 これにより、GUIの操作性を損なわずにバックグラウンドでタイマーを動作させることを実現しました。

　② セキュリティ機能の強化

　　セキュリティ強化メモでは、パスワードをhashlibによるSHA-256でハッシュ化して保存しています。 また、生年月日によるリセット機能を実装し、安全性と利便性を両立しました。

　③ データの永続化

　　jsonファイルを用いて、ToDo・メモ・カレンダーのデータを保存し、アプリ終了後も状態を保持できるようにしました。

　④ UI/UXの工夫

　　ボタンの配色やレイアウトを統一し、直感的に操作できるよう設計しました。 また、フォルダ切り替えやタブ表示など、ユーザーが使いやすい構造を意識しました。

使用技術

  Python Tkinter（GUI） threading（並行処理） hashlib（セキュリティ） json（データ保存）

<筋トレ重量換算アプリ>
概要
  本アプリは、筋力トレーニングにおける「現在の重量と回数」から、1回だけ持ち上げられる最大重量（1RM）を推定し、目標重量に対して必要な回数や不足している重量を算出する計算ツールです。ダークモードを基調とした洗練されたUIが特徴です。

主な機能
  1RM（最大重量）の自動計算: 入力された重量と回数から、即座に最大重量を推定します。

  種目別計算ロジック: スクワット・デッドリフトとそれ以外の種目で計算式を自動で切り替えます。

  目標達成シミュレーター: 目標とする重量を何回挙げればよいか、あるいはあと何kg足りないかをリアルタイムで表示します。

  高解像度（HighDpi）対応: モニターの解像度に合わせて美しく表示されます。

工夫した点
  ① スタイリッシュなUIデザイン（QSSの活用）
    setStyleSheet を用い、CSSに似た記述でデザインを制御しています。

    カラーパレット: 深いネイビー（#0F172A）を背景に、アクセントカラーとしてスカイブルー（#38BDF8）を使用し、モダンなダッシュボード風のデザインを実現しました。

    視覚的フィードバック: 入力フォームにフォーカスが当たると枠線が光る、数値の単位を小さく表示するなど、細かなディテールにこだわっています。

 ② リアルタイム計算の実装
    ボタンを押す手間を省くため、入力内容が変更されるたびに self.calc() が呼び出されるイベント駆動型の設計にしています。
    値が変わるたびに再計算を実行
                 self.ex_in.currentIndexChanged.connect(self.calc)
                 for i in [self.w_in, self.r_in, self.tw_in]:
                 i.textChanged.connect(self.calc)
 ③ 専門的な計算ロジック
  一般的な「O'Conner式」をベースにしつつ、種目の特性に合わせて係数を変更し、より精度の高い推定を行っています。

  計算式: 1RM=重量×回数÷係数+重量

  係数の動的変化: 下半身種目（スクワット・デッド）は 33.3、その他は 40.0 を適用。
使用技術
  Python / PySide6: GUIフレームワークとして、Qt系の高機能なライブラリを採用。

  QSS (Qt Style Sheets): アプリの外観カスタマイズ。

  Exception Handling: 数値以外の入力がされた際のエラー落ちを防ぐ try-except 構造。

実行方法
1.Pythonをインストール

2.必要なライブラリをインストール
  pip install PySide6

3,実行
  python 筋トレ換算表.py
